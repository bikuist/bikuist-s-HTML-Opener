Open HTML from GitHub Raw URL

1. Upload your .html files anywhere in your repo

Background from your repo

Upload .jpg / .png / .webp / .gif files to the images/ folder in your repo.
Add their filenames to the array in the code below → commit & push.
One image is automatically chosen when the page loads.

Refresh the page to get a different random background (if you have multiple images).
